# MakeItWork Frontend

This frontend project is written in React with Vite.

Technologies used:
- React
- Vite
- TypeScript
- ESLint
- Prettier
- Tailwind CSS


## Install and run

To install and run the application, run the following commands:
```bash
npm install
npm run dev
```

Then, the console will tell you which URL to visit to see the frontend running.
